# Github Dashboard

This dashboard is used to visualize the status of our Kyma Projects, and it's focused on tracking external contributions

This is a Vuejs porjects using Github SSO to retrieve a working token.

## Build project

```sh
yarn build
```

## Preview project

```sh
yarn dev
```
